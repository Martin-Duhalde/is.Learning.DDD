# Agent Startup Checklist

Before starting any work in this repository:

1. Read the AI directives located in `docs/ai-directives/`.
2. Review current work orders and their status in `docs/work-orders/board.md`.
3. Consult the latest roadmap and architecture notes under `docs/guides/`.
4. Update or create documentation if new knowledge is produced.

Follow these steps at the beginning of every session.
